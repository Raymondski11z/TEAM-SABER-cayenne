dns masq is used to help network manager. It is needed for you to have internet if you are using network manager (we usually do by default). The port, however, doesn't need to be active. To disable it:

nano /etc/resolv.conf

comment out "nameserver 127.0.0.1

kill -9 "dnsmasq's pid"

dnsmasq -p 0
